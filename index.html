About

Polar.js is a 2D game engine developed in TypeScript for use on the web. It uses WebGL2 to deliver competitive performance and an entity component system to ensure scalability and ease of development.

Features

  • 2D WebGL2 renderer.
  • Entity component system.
  • Instanced rendering.
  • Postprocessing effects.
  • p2.js physics.

Web Usage

Recommended

Add the following line to your HTML document: 
<script src="unpkg.com/polar.js/umd/polar.min.js"></script>
The package is hosted on npm, so can be easily accessed through the UNPKG CDN. For the latest minified version use the example above or specify a semver version: 
unpkg.com/polar.js@1.0.0/polar.min.js
Please refer to the UNPKG site for more details.

Alternate

Alternately, using the node package manager, you can bundle polar.js into your own source code. 
npm install polar.js
